Create Java project using Maven from cmd

mvn archetype:generate -DgroupId=ibm.mq.practice -DartifactId=IBMMQPractice -Dversion=1.0 -DarchetypeArtifactId=maven-archetype-quickstart -DinteractiveMode=false
